# Chimelab（キメラボ）- IRIAM Vライバー事務所

あなたの魅力の研究所「Chimelab（キメラボ）」のWebサイトリポジトリです。
Chimelabは、IRIAMで活動するVライバー事務所であり、所属ライバーは皆なにかの「キメラ」というコンセプトを持っています。

## 🧪 プロジェクト概要

このプロジェクトは、Chimelabの公式サイトおよびランディングページ（LP）のソースコードです。
訪問者に対して事務所のコンセプト、所属ライバーの紹介、サポート体制、応募フォームへの誘導を行います。

## ✨ 特徴

*   **クリエイター運営**: イラストレーター・Live2Dモデラー【LAB:RAT】が運営しており、技術的なサポートが手厚いです。
*   **充実のサポート体制**: 立ち絵プレゼント、収益化審査免除、ランクボーナスなど、ライバーが活動しやすい環境を整えています。
*   **ユニークな世界観**: 「研究所」と「キメラ」をテーマにした独自の世界観を持っています。
*   **サイバーパンクモード**: 隠し機能として、特定の条件でサイトの雰囲気が変わる「サイバーパンクモード」を実装しています。

## 🛠️ 使用技術

*   **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
*   **Backend**: Node.js, Express (静的ファイル配信およびルーティング)
*   **Fonts**: Google Fonts (Kosugi Maru, Noto Sans JP), Font Awesome

## 🚀 始め方

ローカル環境でサイトを確認するための手順です。

### 前提条件

*   [Node.js](https://nodejs.org/) がインストールされていること。

### インストール

プロジェクトのルートディレクトリで以下のコマンドを実行し、依存パッケージをインストールします。

```bash
npm install
```

### サーバーの起動

以下のコマンドでローカルサーバーを起動します。

```bash
node server.js
```

ブラウザで [http://localhost:3000](http://localhost:3000) にアクセスすると、サイトが表示されます。

## 📂 ディレクトリ構成

*   `img/`: ロゴやキャラクター画像などのアセット

*   `apply.html`, `support.html`, `research.html`, `soul.html`: (開発中/予備のページ)
